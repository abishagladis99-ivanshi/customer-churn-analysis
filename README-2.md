
# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data to identify reasons why customers leave a company. It provides insights into churn patterns based on contract type, payment method, and tenure.

## Tools Used
- Excel
- SQL
- R
- Data Visualization

## Dataset
Synthetic dataset created for practice with columns: CustomerID, Gender, Tenure, Contract, MonthlyCharges, PaymentMethod, Churn.

## Key Insight
Month-to-month contract customers have higher churn rates. Insights can help improve customer retention strategies.
